Projects

Smart Links / Instamojo

SmartLinks is a product for businesses through which they can optimize and customize their payment system as required. I was a part of the UX research, prototyping and UI development for the project.

MojoXpress / Instamojo

MojoXpress is a simple fast and reliable shipping service for the small and medium scaled businesses. The service can be used by merchants to ship their products with a minimal fee towards Instamojo. I was involved in the ideation, design and development of the product.

MojoCapital / Instamojo

Developed a byte-sized micro credit system which is provided to micro and small businesses for regularising their cash flow. I was involved in the user interface development and interaction design of the services like Next Day Payout and Instant Payout.

Website Analytics / ZopNow

I pioneered the analytics project of zopnow by creating a simple time-series based analytics dashboard. Given the scale of data and time constraints I decided to use tools like HBase for data storage and Grafana for its feature rich visualisations.

Campaign Management System / ZopNow

Created a distributed microservice to send campaigns to customers. Involved building a coercive API to run campaigns and automate the distribution using Gearman as the distributed Job server.
